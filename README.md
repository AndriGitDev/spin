# SPIN

SPIN is a shareable decision wheel for small group decisions: dinner, games, chores, watch lists, and anything else that gets stuck in chat.

## What changed

- Modern two-panel tool layout with the wheel always visible.
- Shareable URLs that preserve the current question and options.
- Local save and restore for the last wheel.
- Quick-start presets for common decisions.
- Immediate SVG wheel rendering with reliable winner selection.
- Recent pick history, shuffle, fullscreen, and mobile-friendly controls.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Stack

- Next.js 16
- React 19

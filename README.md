# 🎯 SPIN

**Start Picking Instead of Negotiating**

SPIN is a beautiful, interactive spinning wheel web application that helps you make decisions faster. Stop the endless back-and-forth discussions and let the wheel decide!

## Features

- 🎰 **Interactive Spinning Wheel**: Smooth canvas-based animation with a 5-second spin duration
- 🎨 **Beautiful UI**: Gradient backgrounds, animations, and modern design
- 📝 **Customizable Options**: Add up to 12 custom options with your own question/title
- 🎲 **True Randomness**: Each spin uses random rotation for fair results
- ✨ **Visual Effects**: Drop shadows, hover effects, and winner animations
- 📱 **Responsive Design**: Works great on desktop and mobile devices

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
```

### Deploy to Vercel

This project is ready to deploy to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/spin)

Or use the Vercel CLI:

```bash
npm install -g vercel
vercel
```

## How to Use

1. **Enter your question**: e.g., "Where do we eat?"
2. **Add your options**: e.g., "KFC", "Dominos", "Subway"
3. **Spin the wheel**: Click the big spin button
4. **Get your answer**: The wheel spins for 5 seconds and picks a random winner

## Technology Stack

- **Next.js 15**: React framework for production
- **TypeScript**: Type-safe code
- **Tailwind CSS**: Utility-first CSS framework
- **Canvas API**: For smooth wheel animations
- **Vercel**: Deployment platform
